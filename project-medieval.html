One of my first proper forays into environmental art, I wanted to try making something that I felt lent its strengths to the modular kit workflow, so I settled on traditional European medieval architecture with a stylized spin. This project was super fun to work on, and I used a bunch of new skills I'd never tried before! I actually ended making everything seen in this post from scratch aside from the ground textures. One of the main goals was to implement trim sheets into the texture planning. Using them allowed me to reuse a lot of texture area for different surfaces, which saved a lot of time. I wouldn't say the actual sheet arrangements are perfect, but I'm much more comfortable using them now and I'll continue to do so in the future. Another important aspect of this project was to assemble, light & render everything inside of Unreal Engine 4, which was actually a very smooth process due to how I had my kit set up.
